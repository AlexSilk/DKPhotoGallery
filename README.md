### Features
3D Touch
PHAsset

## Requirements
* iOS 8.0+
* ARC
* Swift 3 and Xcode 8

## Installation
#### iOS 8 and newer
DKPhotoGallery is available on CocoaPods. Simply add the following line to your podfile:

```ruby
# For latest release in cocoapods
pod 'DKPhotoGallery'
```
